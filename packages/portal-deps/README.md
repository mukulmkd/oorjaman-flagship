# @oorjaman/portal-deps

Shared workspace dependencies for **admin-web**, **vendor-web**, and **support-web**.

Both `@oorjaman/portal-deps` and `@oorjaman/web-ui` (shell components, Supabase provider, etc.). Keep only app-specific packages in each portal `package.json` (e.g. `recharts` on admin, `country-state-city` on vendor).

```bash
npm install   # from repo root after editing versions here
```
